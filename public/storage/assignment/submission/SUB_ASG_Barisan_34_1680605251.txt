1. Carilah suku ke-100 dari barisan aritmetika 2, 5, 8, 11, …
Jawab:
a = 2

b = u2 – u1 = 5 – 2 = 3

n = 100 un = a + (n – 1)b

un = 2 + (100 – 1)3 = 2 + (99 x 3) = 299

2. Diketahui barisan aritmetika 1, 3, 5, 7, …. un = 225. Tentukan banyaknya suku (n).
Jawab:
a = 1, b = 2, un = 225

un = a (n – 1)b

225 = 1 + (n – 1)2 = 1 + 2n – 2

226 = 2n
